ETRS89-FIN [EUREF-FIN] / GK19FIN + N2000 height

  • WGS84 Bounds: 19.24, 60.08, 19.5, 60.34
  • Scope: Cadastre, engineering survey, topographic mapping (large scale).

  • Axes: Northing, Easting, Gravity-related height (N,E,H). Directions: north, east, up. UoM: metre.
